$('.cl').bind('click', () => {
    console.log($(this)); // ??? 打印出来的不是我点击的对象

愣在那里一会才想起来好像箭头函数会影响 this 的指向 现在想起都觉得自己当时傻 -.-!

问题原因

箭头函数里 this 的绑定是按词法规则来的，不是动态绑定的

箭头函数

  • 箭头函数的出现不是为了让你少写几个单词，而是为了改变 this 的行为
  • 箭头函数总是函数表达式，不是函数声明，而且总是匿名函数
