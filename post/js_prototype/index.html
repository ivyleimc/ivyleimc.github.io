1. 原型链

所有对象都有 [[Prototype]] 内置属性，它指向另一个对象，另一个对象也有，一直到 Object.prototype 为至，这便是原型链

1.1 instanceof

所有函数都有 prototype 属性，用 new 函数调用得到一个对象，它的 [[Prototype]] 指向该函数的 prototype

instanceof 能判断这之间的关系，判断某对象的原型链是否指向某函数的 prototype

1
2
3

function fn() {};
let a = new fn();
a instanceof fn; //true

1.2 isPrototypeOf

使用 Object.create(x) 创建的对象，它的 [[Prototype]] 指向 x

isPrototypeOf 能判断这之间的关系，判断某对象的原型链是否指向另一个对象，我感觉这比 instanceOf 要直接

1
2
3

let obj1 = {};
let obj2 = Object.create(obj1);
obj1.isPrototypeOf(obj2); //true
