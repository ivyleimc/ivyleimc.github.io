BFC 形成条件和应用

(Block Formatting Context) 块格式化上下文

块格式化上下文包含创建它的元素内部的所有内容

创建 BFC 常见的条件

  • float 不是 none
  • position 是 absolute 或 fixed
  • overflow 不是 visible
  • display 是 flex， inline-block，。。。

BFC 的常见作用

  • 清除浮动 （clear:both 也行）

圣杯布局 & 双飞翼布局

  • 使用 float
  • 使用 margin 负值，是两边和中间横向并排
  • 一个用 padding 一个用 margin，使两边与中间不要重叠

定位

relative

依据自身定位

absolutere

依据最近一层定位元素定位
定位元素：absoulute, relative, fixed 或者 body

居中对其 - 水平居中

  • inline 元素：text-aling: center
  • block 元素：margin: auto
  • absolute 元素：left: 50%; margin-left: 负值（要知道元素的 width）
  • absolute 元素：left: 50%; transform: translate(-50%, 0);
  • absolute 元素：left: 0; right: 0; margin: auto;

居中对其 - 垂直居中

  • inline 元素：line-height 的值等于 height 值
  • absolute 元素：top: 50%; margin-top: 负值
  • absolute 元素：top: 50%; left: 50%; transform(-50%, -50%); (水平垂直均居中)
  • absolute 元素：top,left,right,bottom: 0; margin: auto; (水平垂直均居中)
