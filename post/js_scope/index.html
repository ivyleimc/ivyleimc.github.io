作用域 和 自由变量

作用域

  1. 全局作用域
  2. 函数作用域
  3. 块级作用域

自由变量

  1. 一个变量在当前作用域没有定义，但被使用了
  2. 向上级作用域一层层依次寻找，直到找到为止
  3. 如果到全局作用域都没找到，则报错

自由变量的查找，是在函数定义的地方向上级作用域查找，而不是在执行的地方！！！

闭包

函数定义的地方和执行的地方不同
或者说：
作用域应用的特殊情况：

  1. 函数作为参数被传递
  2. 函数作为返回值被返回

重点：如何查找自由变量！！！

    };
}
var f4 = f3();

this

this 的取值是在函数执行时决定的，不是定义时

this 不同场景如何取值

  • 作为普通函数被调用
  • 使用 call，apply，bind 调用
  • 作为对象方法被调用
  • 箭头函数
  • setTimeout/setInterval/DOM事件，等的回调函数里

// 作为普通函数被调用
var a = 123;
function f1() {
    console.log(this);
    console.log(this.a);
}
f1();
// 作为普通函数被调用，this 为 window
//
// 使用 call，apply，bind 调用
var obj = {a: 456};
var obj2 = {a: 789};
var obj3 = {a: 987};
function f2() {
    console.log(this);
    console.log(this.a);
}
f2.call(obj);
f2.apply(obj2);
var f3 = f2.bind(obj3);
f3();
// 使用 call，apply，bind 调用，this 为 我们提供的对象
//
// 作为对象方法被调用
var obj4 = {
    a: 654,
    f4: function() {
        console.log(this);
        console.log(this.a);
    }
}
obj4.f4();
// 作为对象方法被调用， this 为该对象
//
// 箭头函数
var a = 321;
var obj5 = {
    a: 654,
    f5: () => {
        console.log(this);
        console.log(this.a);
    }
}
obj5.f5();
//
var aa = 123;
var obj6 = {aa: 987};
var f6 = () => {
    console.log(this);
    console.log(this.aa);
}
f6.call(obj6);
// 箭头函数，this 为上一层作用域的 this
//
// setTimeout/setInterval/DOM事件，等的回调函数里
var aa = 123;
var obj7 = {
    aa: 654,
    f7: function() {
        setTimeout(function() {
            console.log(this.aa);
        });
    }
}
obj7.f7();
// setTimeout/setInterval/DOM事件，结合了箭头函数
bb = 123;
var obj8 = {
    bb: 654,
    f8: function() {
        setTimeout(() => {
            console.log(this.bb);
        });
    }
}
obj8.f8();
// 再做个看看
cc = 123;
var obj9 = {
    cc: 654,
    f9: () => {
        setTimeout(() => {
            console.log(this.cc);
        });
    }
}
obj9.f9();
